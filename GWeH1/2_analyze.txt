Главная страница состоит из элементов:
    - header - "шапка" - содержит вкладки, лого, информацию о пользователе
    - side bar - вертикальная панель - содержит блоки:
            - основные заголовки
            - инструменты
            - полезные ссылки
            - язык
    - content - центральная часть страницы - содержит блочное отображение избранных по разным критериям статей
    - footer - состоит из элемента footer - содержит информацию по соглашениям и ссылки на другие ресурсы
